NCC SENSOR NODES

Overview

The Ticketing System for NCCC-Sensor Nodes to track node down status of the project "Nationalwide Rollout of Honeypot Threat Capturing Sensors".Tickets will be raised the Node Status tracker of the nodes which are down for more than 12 hours.In case the node gets UP the status will be changed to CLOSED and Ticket will be marked as CLOSED.
