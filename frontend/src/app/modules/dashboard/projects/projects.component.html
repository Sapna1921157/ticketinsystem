NCCC - Sensor Nodes

The project "Nationalwide Rollout of Honeypot Threat Capturing Sensors" Ticketing System is implemented to track the DOWN status of Sensor Nodes. Tickets will be raised the Node Status tracker of the nodes which are down for more than 12 hours. in case the node gets UP in due time before assigning to the analyst, the status will be changed automatically to CLOSED and Ticket will be marked as CLOSED.
